# pnpm - Pragmatic Node PNPM Manager


```zsh
curl -sSL https://raw.githubusercontent.com/robo-monk/pnpm/master/copy%2Bpaste.py | python -
```

# Generate new package

```bash
pnpm create package
```
> and add information via the prompts


# To dev the package

```bash
cd package
pnpm dev
```

# To release the package
```bash
pnpm release
```
