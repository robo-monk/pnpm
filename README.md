# pnpm - Pragmatic Node PNPM Manager


## Install with `zsh`
```
curl -sSL https://raw.githubusercontent.com/robo-monk/pnpm/master/copy%2Bpaste.py | python - && zsh
```

## Generate new package

```
pnpm create package
```
> and add information via the prompts


## To dev the package

```
cd package
pnpm dev
```

## To release the package
```
pnpm release
```
